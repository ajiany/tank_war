# Tank War 全面优化设计

## 概述
对现有坦克大战 MVP 进行全面优化：Bug修复、经典像素风视觉重构、UI重设计、游戏体验增强。

## Phase 1: 关键 Bug 修复
- 敌人坦克开枪逻辑连接
- Bullet.ts 重复位置更新修复
- Input 实例内存泄漏修复
- 每坦克1颗子弹限制

## Phase 2: 经典像素风视觉重构
- 像素矩阵绘制坦克（玩家军绿、敌人多色）
- 砖墙/钢墙/基地像素纹理
- 爆炸动画序列（3-4帧）
- 出生保护盾闪烁动画
- 子弹击墙碎片粒子

## Phase 3: UI 界面重设计
- 右侧信息栏（经典 Battle City 布局）
- 像素字体 HUD
- 经典标题画面 + 坦克光标
- 关卡过渡画面
- Game Over 滚动效果

## Phase 4: 游戏体验增强
- 半格网格对齐移动
- 3种敌人类型（普通/快速/装甲）
- 屏幕微震 + 无敌闪烁
- 3-5关卡系统
- 道具系统（星星/炸弹/铲子/坦克）
